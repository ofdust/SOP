Love Letters

Dearest BASH,

What can I say? If I'm honest with myself, I'll admit that at first I wasn't sure if I'd like you. Being a visual person, I found you difficult, but I love a challenge. You make me feel like I finally have real control over my life my computer.

You're always up for telling me where I am and taking me where I'm going:

pwd - show path to present working directory

cd directoryname - change directories

cd .. - go to parent directory

cd ~ - go to home directory

ls - list directory contents

ls -la - list contents in vertical format with information, including .files

cd - - go to previous directory

And who needs a GUI when together we can do so much ourselves:

touch newfilename.ext - create a new file

nano filename.ext - open file in nano text editor

mkdir directoryname - create new directory

mv filename.ext newdirectoryname/filename.ext - move a directory

mv filename.ext newfilename.ext - rename file

mv directoryname newdirectoryname - rename directory

cp sourcefile.ext destination/copy.ext - copy a file

You even help me get rid of things I don't need in my life anymore:

rm filename.ext - removes files

rmdir directoryname - remove directories

rm -R - to remove a directory and everything in it

Together, BASH, you and I can rule the galaxy computer.

Love,

Shelby

My Darling Git,

Where have you been my whole life? Years of lost ideas, broken code, when all I needed was a free download and a few short commands:

git init - identify current folder as a git repository (for our love)

git add filename.ext - add file to staging area (of love)

git add . - go wild and add all files with changes to staging area

Your commitment to me is unceasing:

git commit -m "short message about changes" - creates a record of the current state of our love the project

Together we'll create so many memories to look back on:

git log - list previous commits

git checkout hashnumber - brings you back to this commit

And go on many journeys together:

git branch newbranchname - create a new branch

git branch - lists all branches (branch with the * is the current branch)

git checkout branchname - switch branches

git merge branchtobemerged - merge a branch with current branch

git branch -m oldname newname - change branch name

git branch -D branchname - delete branch (but not our love)

Oh, git, where would I be without you?

Love,

Shelby
